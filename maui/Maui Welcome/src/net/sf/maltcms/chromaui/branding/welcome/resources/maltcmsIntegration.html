Maltcms Integration

Maltcms is already contained within Maui, however, in order to edit and run custom pipelines, you will need to install a version of Maltcms. After the installation, you can can import the available pipelines into a project and execute them immediately from the project's context menu (available with right-click).

  1. Install Maltcms

Customizing Maltcms Pipelines

Maltcms pipelines are visible after installation and pipeline import in the File tab of the project explorer. Type CTRL+2 to open the file tab, or select Window->Files in the main menu.
