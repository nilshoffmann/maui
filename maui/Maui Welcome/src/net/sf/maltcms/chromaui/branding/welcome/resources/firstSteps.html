First Steps with Maui

After downloading and installing Maui for your operating system, there are only a few additional post-installation steps necessary. After performing these all features of Maui will be available for use. The statistical analysis methods in Maui use GNU R as their backend. Thus, you need to install a recent release of GNU R on your local computer or on a server machine within your local network that is accessible from your local computer.

  1. Install GNU R
  2. Start up R (preferrably a 64 bit session as Administrator
  3. At the command prompt, type install.packages("Rserve") and hit Enter
  4. Select a download mirror and install the package
  5. At the command prompt, type library(Rserve); Rserve() and hit Enter
  6. On Windows, a dialog may pop up asking you to allow Network connections for Rserve on your private network. You need to allow the default selected option.
  7. Check your R and Rserve installation
  8. Close R
  9. Check your R and Rserve installation again. Maui should start up a new local instance of Rserve.
  10. Install Maltcms

Should you encounter any problems during the installation, please contact the authors and report what you tried to do together with Maui's output (View->IDE Log).
