Features

25 Oktober 2014

1D Chromatography

+ 1D intensity profile, side and top view + 1D extracted ion profile + 2D intensity profile (heatmap)

2D Chromatography

+ 1D intensity profile, side and top view + 2D intensity profile (heatmap) + 3D intensity profile (height mapped heatmap)

Mass Spectrometry

+ MS database import from msp format + database search with cosine score and retention index + ms view

Peak Data

+ Maltcms peak format (TIC,EIC) + LECO ChromaTof © csv peak export format + GC Image © Blob csv peak export format

Statistics

+ 2D fold change plot + 3D PCA plot + visualization of tabular data, e.g. ANOVA reports

File Formats

ANDI-CHROM, ANDI-MS, netCDF, mzML, msp (NIST AMDIS © mass spec format).

Interactivity

+ synchronization of viewports + overview/detail views of chromatograms, peaks and peak groups

Integration and Extensibility

+ Export of mass spectra to clipboard, compatible to GMD database input format + Groovy scripts for rapid prototyping of custom processing pipelines + Rserve / Rscript integration + NetBeans RCP Plugin System for extension of Maui's capabilities
