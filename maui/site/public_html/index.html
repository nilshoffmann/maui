Welcome to Maui

an Open Source application for chromatography-mass spectrometry using (GC/LC)n-MS instruments based on Maltcms.

Maui is a modular Netbeans Rich Client Platform application, which allows for easy extension of its functionality, while still maintaining platform independence.
