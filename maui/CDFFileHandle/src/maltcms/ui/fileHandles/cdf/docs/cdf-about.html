About Maui IO netCDF

Maui uses the as its primary data format for intermediate results. Futhermore, it supports chromatography and mass spectrometry data stored in netCDF files, following the ANDI-CHROM and ANDI-MS standards.

netCDF Navigator view

The navigator view may be opened via 'Window->Navigator'. If you select a netCDF file node in the project explorer, the navigator view will display a hierarchical representation of the file.

Charting netCDF files

Right-clicking on a netCDF file node in the project explorer opens the associated context menu. Select 'Open->Open as chart' to show the variable selection dialog. This allows to create quick charts of data contained in the netCDF files.
