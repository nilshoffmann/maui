Custom Pipelines

Maui allows the definition of custom Maltcms pipelines.

  1. Install Maltcms
  2. Open a project (optional) using File->Open Project
  3. Right click on your project in the Projects view and select Import Maltcms Pipelines
  4. Select and open the pipelines folder in the project tree view at Window->Files
  5. Browse to any of the spring xml files, open and edit
  6. The corresponding .mpl file defines further runtime parameters
