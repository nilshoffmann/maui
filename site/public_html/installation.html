Installation of Maui

Download one of the following installers suitable for your operating system and run it to install Maui on your system.

Release History

Maui 1.2.1

Released on April 24th, 2013

Requirements
  • Maui 1.2.1 requires a Java Runtime Environment (JRE) 7 in order to run. We recommend to install a 64 bit version of Java where possible to allow more memory to be allocated.
  • The statistical backend of Maui is based on R and accessed using Rserve. Please consult the respective sites on notes how to install R for your operating system.
    Rserve may be installed from within a running R session via install.packages("Rserve").
New Features
  • Peak overlays are now better visible for 1D chromatogram views
  • Peak overlays are colored according to peak descriptor container color, or by treatment group color
  • Added a diff button to mass spectral viewer to show a difference spectrum of two mass spectra
  • Peak groups now show a percentage of covered chromatograms
Bug Fixes
  • Fixed Rserve connection testing on Mac OS X related to the cairo library
  • Fixed invalid cache paths for Ehcache on Windows
  • Fixed EIC display options for 1D chromatogram views
  • Fixed label display of masses with highest intensities in mass spectral view
  • Changed update center locations, SF FRS location is incompatible with NetBeans updater
Known Issues
  • Retention time to peak index mapping for ChromaTOF data is disfunctional when importing peaks additionally to raw data.
Installers
  • Windows
  • MacOS X
  • Linux
  • Solaris

Maui 1.2

Released on March 26th, 2013

Requirements
  • Maui 1.2 requires a Java Runtime Environment (JRE) 7 in order to run. We recommend to install a 64 bit version of Java where possible to allow more memory to be allocated.
  • The statistical backend of Maui is based on R and accessed using Rserve. Please consult the respective sites on notes how to install R for your operating system.
    Rserve may be installed from within a running R session via install.packages("Rserve").
New Features
  • Welcome center to aid with first steps in Maui
  • Viewport synchronization for chromatogram views
  • Selection system for chromatogram views
  • Improved integration and configuration of external tools
  • Import of ChromaTOF peak lists as pseudo-chromatograms
  • Completed integration of Anova and PCA via RServe
  • Interactive visualization of PCA results
